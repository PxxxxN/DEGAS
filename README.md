## Installation
* Step 1 Install python3, pip3, and Tensorflow
```bash
pip3 install tensorflow==1.4
```

* Step 2 Install devtools
```R
install.packages("devtools")
```
* Step 3 Install DEGAS
```R
library(devtools)
install_github("tsteelejohnson91/DEGAS")
```
